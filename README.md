# Jobportal-scraper

<p>Web Scraping is the process of automatically collecting data from websites using programs or scripts.
Instead of manually copying information, web scraping tools extract data directly from web pages and store it in structured formats like CSV, Excel, or databases.</p>

## Advantages of Web Scraping 
- Saves time and effort
- Automates data collection 
- Enables data analysis and insights
- Works at large scale


## Libraries used for webscraping
- BeautifulSoup – Parses HTML content
- Requests – Sends HTTP requests
- Scrapy – Large-scale scraping framework
- Selenium – Browser automation tool
- Pandas – Data storage and analysis

## What is Webdriver?
- WebDriver is a tool that controls a web browser automatically

## Why WebDriver?

- Automates browser actions
- Extracts data from dynamic websites
- Used in:
    - Web scraping
    - Automation testing
    - Bots

## Path Uses 
- `By.XPATH`: Finds elements by their structure or text in HTML.
- `By.CSS_SELECTOR`: Finds elements by class, id, or CSS rules.
